== Description ==

This plugin provides Persian bbPress with Jalali date and some CSS styles. It also changes the default editor of bbPress and adds a user related menu bar above the forums and topics pages.

Persian language pack for bbPress is also included in this plugin.

== Installation ==

1. Download and unzip the plugin. Then upload the `"persian-bbpress-tools"` folder to the `/wp-content/plugins/` directory, or you can simply install it using WordPress plugin installer
2. Copy the `"bbpress"` folder within the root directory of plugin into the `/wp-content/languages/` to make bbPress use the Persian language pack
3. Change the default date format of WordPress to `"j F Y"`
